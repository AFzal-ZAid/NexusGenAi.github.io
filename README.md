AI Nexus — Intelligent AI Assistant Platform
🚀 Overview

AI Nexus is a modern AI-powered full stack web application built using Spring Boot, Spring AI, and React.js.
The platform provides an interactive AI experience where users can:

💬 Ask AI questions
🎨 Generate AI images
🍽 Generate AI-based recipes
⚡ Experience a modern futuristic UI
