# 🚀 NexusGen AI

An advanced AI-powered SaaS platform inspired by modern conversational AI systems like ChatGPT and iAsk AI, built using **React**, **Spring Boot**, and **Generative AI APIs**.

NexusGen AI combines intelligent chat assistance, AI image generation, and AI-powered recipe creation into one seamless modern web platform.
